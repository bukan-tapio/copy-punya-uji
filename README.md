## App pembayaran SPP
Ini adalah Aplikasi laravel pertama saya sekaligus tugas UKOM :)

## Tech yang digunakan

- Laravel 9
- Livewire
- Tailwindcss
- Alpine js

## Instalasi

- composer update
- npm install
- php artisan migrate
- php artisan db:seed

## Fitur 

- Login MultiUser
- CRUD Data Siswa
- CRUD Data Petugas
- CRUD Data Kelas
- CRUD Data SPP
- CRUD Data Entri
- History
- Generate Laporan Excel Dan Pdf
- Logout
